Repositorio de aplicaciones para Yupp PHP Framework.

En este sitio publicaremos todas las versiones de las distintas aplicaciones y las futuras aplicaciones que desarrolle cualquier miembro de la comunidad Yupp.

Aquí pueden ver las aplicaciones que ya están disponibles: http://code.google.com/p/yupp-apps/downloads/list

Para instalar una aplicación, simplemente es descargar el ZIP con una aplicación y descomprimir su contenido en la carpeta "apps" de Yupp (carpeta que existe desde la versión 0.2.2 del framework).

Descarga ahora Yupp PHP Framework: http://code.google.com/p/yupp/


_Nota: la aplicación "movix" depende de una API externa de IMDB, por lo que puede dejar de funcionar sin previo aviso si se hacen cambios en IMDB._
